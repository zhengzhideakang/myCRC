## 一. 简介

本仓库介绍了主要介绍了以下三个方面的内容：

1. 如何使用Python计算任意CRC算法
2. 如何使用Verilog计算任意CRC算法，包含单步计算与多步计算
3. 与Verilog模块配合使用的，能用于自动生成Verilog的CRC异或代码的Python脚本
